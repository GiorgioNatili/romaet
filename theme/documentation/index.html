Installation

Installation of our template is really easy, you just need to make a few steps to get it done.

1. Cache

Recolorline the cache. We recommend you to disable it at all until magento is not configured.

2. Upload Files

Our theme is compatible with latest (for now 1.7.0.0) version of magento community.

Using your favorite FTP manager upload folders app and skin of theme directory to root of your magento. No one magento default file will be overridden.

3. Re-login

We provide 3 additional modules with the theme and you have to log out and log in again for correct installation all of them.

4. Set Package

Go to System/Configuration/Design page of your admin panel and set Current Package Name as meigeetheme

That's it. Template is installed.

Or you can use the demo data that included in this package to get the same view as on the demo.

To get the same view as on the demo just follow steps below:
  • Upload sources/demo/media folder to the root directory of your magento
  • Import sources/demo/dump.sql.gz into your database
  • Install magemto using this database

Please notice: Dump file must be imported into your database prior magento installation

Retina Ready Options

"Retina Ready" feature is enabled by default but you need follow a few steps to make your own content be retina ready too.

  1. To use your own images in content you should use following code: 
    <img src="path/to/image" data-srcX2="path/to/retinaready/image">
  2. If you want to add some other languages you need to put your own retina ready images of flags into skin/frontend/meigeetheme/default/images/@x2/ and name them as "English@x2". The same step you need to follow for currency images.
  3. Upload your own retina ready logo into "Meigee -> Theme Options -> General Options -> Custom Logo" tab

Static Blocks

Magento allows you to create blocks of content that you can use throughout your Web Store. We have created a few statics blocks to give you ability to put your own content into the most important regions of the store.

How to create a static block?

Go to CMS/Static Blocks and click the button Add New Block and follow steps below:

  1. Enter your Block Title
  2. Enter block Identifier. List of predefined you can find below.
  3. Store View set as All Store View or specify to which store view you want a new block apply
  4. Set Status as enabled to display a block in frontend
  5. Put your content into field Content. Notice: You can use Meigee Widgets in any static block.
  6. Click the button Save Block

Predefined Static Blocks

  1. custom_product_tab_XX - custom tab for products. To create additional tab for a product just create a static block called 'custom_product_tab_XX' where XX - product id, and fill it with your own additional content
  2. custom_product_tab2_XX - second custom tab for products. To create additional tab for a product just create a static block called 'custom_product_tab2_XX' where XX - product id, and fill it with your own additional content
  3. cart_banner - block on a product page
  4. custom_footer - custom block in the footer
  5. custom_footer_2 - custom block in the footer
  6. likes
  7. slider_2_text_box
  8. social_links

Template Versioning

When you buy our template you can be sure that it will be updated to the latest version of magento after it will be released and all bugs will be fixed shortly.

We know that our clients make a tone of customization work in the template and don't want just to delete old version and use newer one or replace some of changed files. They want to implement fixes and changes into their customized template

We had a deal with it many times, so decided to impleemnt SVN into our work to get our client an ability to implement changes and fixes into their templates. Let's take a look at updating template procces

When to see if new version is available?

Follow us in tvitter to get updates first.

Every release of template is accompanied with description what we done.

How to learn if new version of template is available on themeforest?

Every new version of template must be reviewed by envato staff. When it done thumbnail in template changes and you can see it on your downloads page

How to learn version of your template?

Just open up skin/frontend/meigeetheme/default/css/styles.css in your favorite code editor. At the top of this file you can see a line of code with version of template (Example: @template_version 1.0)

How to update template to the latest version?

First of all go to your downloads page on the themeforest and download the latest version of template. We don't send files directly via email.

If you haven't done any changes in the template just replaced old version with new one.

If you have done some changes in code you should go to a folder changes of the downloaded package where you can find .diff files with all changes

FAQ

How to disable links from footer

About Us, Customer Service, Privacy Policy
CMS -> Static Blocks -> Footer Links

Site Map
System -> Configuration -> Catalog vertical tab -> Search Engine Optimizations: Autogenerated Site Map set as "disable"

Search Terms
System -> Configuration -> Catalog vertical tab -> Search Engine Optimizations: Popular Search Terms set as "disable"

Advanced Searchm, Orders and Returns
Open up app/design/frontend/meigeetheme/default/layout/local.xmlin your favorite text editor and uncomment following code

<reference name="footer_links">
	<action method="removeLinkByUrl"><url helper="catalogsearch/getAdvancedSearchUrl"/></action>
	<remove name="return_link" />
</reference>
Contact Us
System -> Configuration -> Contacts vertical tab -> Contact Us: Enable Contact Us set as "no"
